La etiqueta de párrafo es la etiqueta p. Si queremos introducir un salto de línea ponemos donde queremos la etiqueta br

Como por ejemplo
entre esta línea 1
y esta línea 2

Y si queremos pasar una línea horizontal a nuestro texto ponemos la etiqueta hr

para mostrar una línea horizontal

También podemos darle atributos a la línea con el atributo align alineado y width ancho
